# Createverse Modpack
[Download through Modrinth](https://modrinth.com/modpack/createverse-modpack)

Explore endless possibilities in CreateVerse, a versatile modpack for all playstyles!

Createverse is a modpack built on NeoForge 1.21.1, designed for the CreateVerse SMP community by [VoltCore Network](https://discord.com/invite/E8q2tc2jz7). Dive into an exciting world featuring the Create mod and its variety of intriguing addons, Farmer's Delight, Continents, BOP, CC: Tweaked, AE2, Create: Northstar, among many others.

This modpack offers many quests and a diverse experience without a specific theme
